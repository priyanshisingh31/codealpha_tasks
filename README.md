# CodeAlpha Java Projects 

This repository contains multiple Java console-based projects developed as part of the CodeAlpha Internship Program.

---

##  Projects Included

### 1️ Student Grade Tracker
A console-based application to input, calculate, and display student grades including average, highest, and lowest scores.

File: StudentGradeTracker.java

---

### 2️ Stock Trading Platform
A simulated stock trading system allowing users to buy/sell stocks and track portfolio performance.

File: StockTradingPlatform.java

---

### 3️ AI Chatbot
A simple Java chatbot that interacts with users through predefined responses in the console.

File: AIChatbot.java

---

### 4️ Hotel Reservation System
A hotel booking system with room categories, booking, cancellation, payment simulation, and file handling.

File: HotelReservationSystem.java

---

##  How to Run the Program

### Step 1: Clone Repository
git clone https://github.com/priyanshisingh31/codealpha_tasks.git

### Step 2: Go to Folder
cd codealpha_tasks

### Step 3: Compile Java File
javac FileName.java

### Step 4: Run Java File
java FileName

### Example:
javac HotelReservationSystem.java
java HotelReservationSystem
